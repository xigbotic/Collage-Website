Introduction: You're Welcome.

Collage Bot is the only Discord bot you need. It handles music stuff and server moderation because we couldn't be bothered to make two bots. We took premium features, stuffed them into this thing, and made it free. Why? Because we can. Get to tracking your music collection and using features you don't deserve.

ATTENTION IDIOTS: Yes, Collage is in active development. That means things change. Often. Don't complain when a command breaks; we're making it better. You're just a beta tester who pays nothing. Adapt.

How to Use (The Simple Part)

If you need instructions for these three simple steps, maybe Discord isn't for you. But fine, here they are:

  1. Go click the button in the header. (Yes, the one that says 'Invite').
  2. Use the highly complex @collage command prefix to access features.
  3. Type @collage help if you immediately forget everything you just read.
