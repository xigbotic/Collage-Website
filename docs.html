Collage Bot Documentation

This page will contain all the technical documentation you need to use and integrate our services.

Introduction

Welcome to the documentation! Collage Bot is a versatile Discord bot that provides many premium commands and features, for completely free. It helps you track your music collection and enjoy seamless integration with your server.

Please note that Collage is still in its development stages, with new commands and features being added every day. The current moderation commands will only work if you are on the whitelist, which is not a automatic thing. These features will be Implemented at a later date for everyone to use. The premium features are marked with a ⭐ and can only be used in our main Discord server. This will also change at a later date so if you have the premium role in our Discord server, you will be able to use premium features everywhere.

How to Use

Follow these steps to get the service up and running:

  1. Invite Collage Bot to your server.
  2. Use the - command prefix to access features.
  3. Type -help to see available commands.

Commands

Command Description Example
-ban Ban a user from the server (whitelist). -ban @xbox360s
-kick Kick a user from the server (whitelist). -kick @xbox360s
-lock Locks the current channel. -lock
-unlock Unlocks the current channel. -unlock
-qp Creates a poll on your message. -qp am i cool
-afk Set a custom AFK message. -afk im sleeping
-embed Create a custom embed (whitelist). -embed
-purge Deletes a specified amount of messages from the channel (whitelist). -purge 99
-bc Clears bot messages. -bc
-si Displays server information. -si
-ping Provides bot latency. -ping
-leak Provides song file and information for a song. -leak rental
-snippet Provides known snippets of a provided song. -snip k like a russian
-juiceinfo Provides all known information on a provided song. -juiceinfo push me away
-juicepic Provides up to 10 random pictures of Juice WRLD. -juicepic 10
-session Provides a session edit for a specified song, if available. -session nintendo
-juicegif Provides up to 10 random GIFs of Juice WRLD. -juicegif 7
-stem Provides a specified stem (Not All Supported). -stem violence
-remaster Provides a specified remaster. -remaster timezone
-comp ⭐ Provides a user with an updated Mega comp in the user's DMs. -comp
-date ⭐ Provides all leaks from a specified date. -date 8/22/24
-leaktracker ⭐ Sends all dated leaks to the user's DMs. -leaktracker
