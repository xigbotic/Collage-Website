Changelog

Here you can find all the updates and changes made to Collage Bot.

Version 2.0.3 - July 10, 2025

  • Fixed numerous issues involving the -leak command.
  • Improved UI for the -remaster command.
  • Improved UI for the -stem command.
  • Improved UI for the -leak command.
  • Improved UI for the command.
  • Fixed an issue where the -ban command would function incorrectly.
  • Fixed an issue where the -jail & -unjail command would function incorrectly.
  • Fixed an issue where the -help command would not provide the full command list.
  • Added a warning system.
  • Improved the -ui command.
  • Added a -setjailrole command.
  • Added a -jailsetlog command.
  • Added a -jailsetjail command.
  • Added a filter moderation system.
  • Added a -8ball command.
  • Removed the -sob command due to technical issues.
  • Removed the -skull command due to technical issues.
  • Improved numerous other quality of life features.

Version 1.2.7 - April 4, 2025

  • Fixed a issue where the -stem command would return a error.
  • Fixed a issue where the -remaster command would return a error.
  • Added a -freestyle command to get any studio recorded freestyle.
  • Added a -era command to get the era of any track.
  • Gave the -help command a new look, allowing for easier navigation.
  • Added a -grail command with many subcommands, check docs for info.
  • All moderation commands now have the ability to be executed as long as you have the required permissions.

Version 1.2.6 - March 23, 2025

  • Fixed an issue that caused the -leak command to not look for multiple versions of a song.
  • Added a -producer command (Premium Feature)
  • Added multiple grail commands. Check out docs for more info.
  • Added a -ttt (toptentracks) command for Last.Fm.
  • Added a -convert command to convert any Youtube link into a MP3. (Premium Feature)

Version 1.2.5 - March 20, 2025

  • Fixed an issue where -snippet would send a unknown error.
  • Finished support for -uzileakcommand. (Premium Feature)
  • Added economy with multiple different commands involving it. Check out docs for info.
  • Added 42 new snippets to the -snippet command.
  • Improved error handling.
  • Fixing a issue thats causing the bot to not look for multiple versions of a song.

Version 1.2.4 - March 12, 2025

  • Added support for Last.FM commands and features.
  • Support for Uzi leaks and info in development (Premium Feature)
  • Fixed a issue where premium commands could only be excuted in WRLD of the Perced.

Version 1.2.3 - March 10, 2025

  • Added -remaster command for remastered tracks.
  • Improved response time for -leak command.
  • Fixed a bug causing -session to return incorrect data.
  • Added -stem command to get stems from tracks.
  • Fixed a issue where -leak didn't return output in low file size servers.

Version 1.2.2 - February 25, 2025

  • Added new command -leaktracker (Premium feature).
  • Updated bot invite permissions for better security.
  • Bug fixes and performance improvements.

Version 1.2.1 - February 10, 2025

  • Added support for up to 10 images in -juicepic.
  • Fixed an issue where -ping showed incorrect latency.
  • Backend optimizations for faster command processing.
