Changelog

Here you can find all the updates and changes made to Collage Bot.

Version 1.2.4 - March 12, 2025

  • Added support for Last.FM commands and features.
  • Support for Uzi leaks and info in development (Premium Feature)
  • Fixed a issue where premium commands could only be excuted in WRLD of the Perced.

Version 1.2.3 - March 10, 2025

  • Added -remaster command for remastered tracks.
  • Improved response time for -leak command.
  • Fixed a bug causing -session to return incorrect data.
  • Added -stem command to get stems from tracks.
  • Fixed a issue where -leak didn't return output in low file size servers.

Version 1.2.2 - February 25, 2025

  • Added new command -leaktracker (Premium feature).
  • Updated bot invite permissions for better security.
  • Bug fixes and performance improvements.

Version 1.2.1 - February 10, 2025

  • Added support for up to 10 images in -juicepic.
  • Fixed an issue where -ping showed incorrect latency.
  • Backend optimizations for faster command processing.
